# Basic Python Programs

## Introduction

This repository contains beginner-level Python programs created and executed using Google Colab. These programs help in understanding the basic concepts of Python programming such as printing output, checking Python version, using conditional statements, and writing multiple statements.

## Tools Used

* Python 3.12.13
* Google Colab
* GitHub

## Programs Included

### 1. Hello World Program

```python
print("Hello,World!")
```

### Output

```python
Hello,World!
```

---

### 2. Python Version Program

```python
import sys
print(sys.version)
```

### Output

```python
3.12.13 (main, Mar 4 2026, 09:23:07) [GCC 11.4.0]
```

---

### 3. If and Else Statement

```python
if 5 > 2:
    print("Five is greater than two!")
else:
    print("Five is not greater than two!")
```

### Output

```python
Five is greater than two!
```

---

### 4. Multiple Print Statements

```python
print("Hello World!")
print("Have a good day.")
print("Learning Python is fun!")
```

### Output

```python
Hello World!
Have a good day.
Learning Python is fun!
```

---

### 5. Multiple Statements in One Line

```python
print("Hello"); print("Hey hey"); print("Bye bye!")
```

### Output

```python
Hello
Hey hey
Bye bye!
```

---
