# Program 1: Print on the Same Line

## Code

```python
print("Hello World!", end=" ")
print("I will print on the same line.")
```

## Output

```python
Hello World! I will print on the same line.
```

## Explanation

The `end=" "` parameter prevents the next `print()` statement from starting on a new line. Both outputs are displayed on the same line with a space between them.

# Program 4: Comments in Python

## Code

```python
#This is comment
print("This is comment")    #This comment

"""
This is comment
for more than one line
"""

print("This is comment")
```

## Output

```python
This is comment
This is comment
```

## Explanation

* `#` is used for single-line comments.
* Inline comments can be written beside statements.
* Triple quotes `""" """` are used for multi-line comments.
* Comments are ignored during execution.
