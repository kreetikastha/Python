## Introduction

# Theory of Python Variables

## What is a Variable?

A variable is a container used to store information or data values in memory. Each variable has a name and holds a value that may change during execution.

Python automatically assigns the data type according to the value stored in the variable.

---

## Creating Variables

In Python, variables are created when a value is assigned to them. There is no need to declare the variable type separately.

Variables can store different types of data such as numbers, text, and Boolean values.

---

## Variable Naming Rules

The following rules are used while naming variables in Python:

* Variable names must start with a letter or underscore `_`.
* Variable names cannot begin with numbers.
* Variable names are case-sensitive.
* Spaces and special characters are not allowed.
* Variable names can contain letters, digits, and underscores.

---

## Types of Data Stored in Variables

| Data Type | Description                     |
| --------- | ------------------------------- |
| Integer   | Stores whole numbers            |
| Float     | Stores decimal numbers          |
| String    | Stores text or characters       |
| Boolean   | Stores `True` or `False` values |

---

## Casting Variables

Casting means converting one data type into another. Python allows variables to be converted into integer, float, or string types whenever needed.

Casting improves flexibility and helps in performing calculations and formatting outputs.

---

## Multiple Variable Assignment

Python allows assigning values to multiple variables at the same time. Multiple variables can also store the same value.

This feature reduces code length and improves readability.

---

## Output Variables

Variables can be displayed using the `print()` function. Text and variables can also be combined together to create meaningful output.

---

## Variable Scope

The scope of a variable determines where the variable can be accessed in a program.

### Local Variables

Local variables are created inside functions and can only be used within that function.

### Global Variables

Global variables are created outside functions and can be accessed throughout the program.

---

* No need for separate type declaration.
* Can store different data types.
* Values can change during execution.
* Support multiple assignments.
